Explicação das Tags em HTML Passadas Pelo Professor Ícaro Botelho



  1. < p > TAG de paragráfo. < / p >

  2. < h1 até h7> TAG de Título. < / h1 até h7>

  3. < html > TAG para Iniciar a Estrutura de Código em HTML. < / html >

  4. < head > TAG para colocar elementos de configuração e Titulo do Site irão aparecer lá. < / head >

  5. < body > TAG onde irão aparecer todo o corpo do site. É dentro desta TAG que serão carregadas as outras TAG de HTML. < / body >

  6. < hr > TAG de separeção, coloca uma linha, como você pode ver olhando logo na Parte de cima deste site. < / hr >

  7. < br > TAG de quebra de linha. < / br >

  8. < div > TAG onde irá agrupar elementos do site. < / div >

  9. < section > TAG semântica, representa uma seção, onde pode haver DIV com conteúdos. < / section >

  10. < footer > TAG onde ficará a parte do rodapé do site. < / footer >

  11. < header > TAG onde ficará a parte do Cabeçalho. < / header >

  12. < ul > TAG de lista não ordenada. < / ul >

  13. < li > TAG onde ficarão os conteúdos da lista. < / li >





10 Significados de TAG em HTML Diferêntes



  1. < strong > TAG de texto. Deixa a Letra em Negrito. < / strong >

  2. < a > TAG de Link. Click aqui para Visitar o Google < / a >

  3. < img > TAG para adicionar Imagens. < / img >

  4. < i > TAG para colocar textos em Itálico. < / i >

  5. < ol > TAG lista Ordenada. < / ol >

  6. < legend > TAG usada normalmente junto com formulários, onde ela cria um rótulo para o conteúdo. < / legend >

  7. < input > TAG para Inserir algum valor. Existem vários tipos: (type="text"; type="password"; type="range"). < / input >

  8. < em > TAG onde irá dar Ênfase. < / em >

  9. < form > TAG define a criação de um Formulário. < / form >

  10. < botton > TAG onde é criado um Botão. < / botton >
